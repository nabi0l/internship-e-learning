2. Introduction to HTML and CSS Basics

HTML and CSS are fundamental for web development. HTML structures the content of a webpage using elements like headings, paragraphs, links, and images, while CSS enhances its appearance by defining styles such as colors, fonts, spacing, and layouts. HTML ensures content is well-organized and meaningful, while CSS makes it visually appealing and responsive across different devices. Together, they create structured, stylish, and user-friendly web pages.

2.1 HTML Basics

HTML (HyperText Markup Language) is a standard markup language used to create web pages. It describes the structure of a document, including the document's title, headings, paragraphs, images, hyperlinks, and more. HTML documents consist of HTML tags, which define the content and structure of the page.

HTML Structure

HTML elements are the building blocks of an HTML document. They are represented by tags enclosed in angle brackets (<>) and consist of a start tag, content, and end tag. The content inside the tags defines the content of the element. Here are some common HTML elements:

  • <html> - The root element of an HTML document
  • <head> - Contains meta information about the document
  • <body> - Contains the visible content of the document
  • <h1> - Defines a large heading
  • <p> - Defines a paragraph
  • <img> - Defines an image
  • <a> - Defines a hyperlink
