4 Introduction to JavaScript Basics

JavaScript is a versatile, cross-platform, and object-oriented programming language. It's used to create interactive web pages, web applications, and mobile apps. In this module, we'll explore the basics of JavaScript, including variables, data types, conditionals, loops, functions, and objects.

4.1 Variables

Variables are used to store and manipulate data values. They have a name, a type, and a value. In JavaScript, variables are declared using the `let` keyword, followed by the variable name and an initial value. Here's an example: 

let x = 10;

Data Types

JavaScript has several data types:

  • Numbers- (e.g., 10, 3.14, -5)
  • Strings- (e.g., "Hello", 'World')
  • Booleans- (true or false)
  • Null- (null value)
  • Undefined- (uninitialized value)
  • Symbol- (new in ES6)
  • Objects- (e.g., { name: "John", age: 30 })

4.2 Conditionals & Loops

Conditionals

Conditionals are used to control the flow of a program based on certain conditions. JavaScript supports the following conditional statements:

if

The if statement evaluates a condition and executes the associated code block if the condition is true.

if...else

The if...else statement evaluates a condition and executes one of the associated code blocks based on the condition.

else if

The else if statement is used to check multiple conditions and execute the associated code block if any of the conditions are true.

Here's an example of using an if statement: 

                         let x = 10;
                         if (x > 0) {
                             console.log("x is positive");
                         } else {
                             console.log("x is not positive");
                         }

4.2.1 Switch Statement

The switch statement executes different actions based on conditions. It's often used with strings or numbers. 

                         let x = 1;
                         switch (x) {
                             case 0:
                                 console.log("x is 0");
                                 break;
                             case 1:
                                 console.log("x is 1");
                                 break;
                             default:
                                 console.log("x is not 0 or 1");
                         }

4.2 Loops

For Loop

The for loop is used to iterate over a block of code a specified number of times. It has three parts: initialization, condition, and increment/decrement. Here's an example: 

for (let i = 0; i < 5; i++) {
    console.log(i);
}

While Loop

The while loop is used to iterate over a block of code as long as a specified condition is true. It has a single condition. Here's an example: 

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

Do...While Loop

The do...while loop is similar to the while loop, but it executes the loop at least once before checking the condition. Here's an example: 

            let i = 0;
                    do {
                console.log(i);
              i++;
           } while (i < 5);

4.3 Functions

Functions are reusable blocks of code that perform a specific task. They can take input parameters, perform calculations, and return a result. Here's an example of a function in JavaScript: 

function greet(name) {
    console.log("Hello, " + name);
}
greet("John");

4.4 Objects & Arrays

4.4.1 Objects

Objects are used to store related data and behavior in a single unit. They are created using the {} syntax and can have properties and methods. Here's an example of an object in JavaScript: 

let person = {
    name: "John",
    age: 30,
    greet: function() {
        console.log("Hello, " + this.name);
    }
}
person.greet();

4.4.2 Arrays

Arrays are used to store multiple values in a single variable. They are created using the [] syntax and can contain any data type. Here's an example of an array in JavaScript: 

let numbers = [1, 2, 3, 4, 5];
console.log(numbers[0]); // Output: 1
              console.log(numbers[4]); // Output: 5
